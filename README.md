# Excel Merge and Analysis Project

A project developed to merge multiple Excel datasets into a single consolidated file and perform Exploratory Data Analysis (EDA) for data understanding and insights generation.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* OpenPyXL
* R-studio

## Features

* Excel File Merging
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Data Visualization
* Summary Report Generation
